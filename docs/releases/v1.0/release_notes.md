We are thrilled to announce the first stable release of **FluxLinux**! Version `1.0` brings a complete, hardware-accelerated Linux desktop experience to your Android device.

## 🚀 What's New in v1.0

| Category | Features |
|:---:|---|
| **Core** | • One-tap Debian installation (PRoot & Chroot)<br>• Hardware acceleration setup wizard<br>• Support for rooted and non-rooted devices |
| **Graphics** | • GPU acceleration with **Turnip** and **VirGL**<br>• Termux:X11 integration for smooth GUI |
| **Desktop** | • Pre-configured **XFCE4** desktop<br>• Beautiful Material 3 UI with glassmorphism<br>• Custom "Space" themes included |
| **Dev Stacks** | • **Web**: Node.js, React, VS Code<br>• **Mobile**: Flutter, Kotlin<br>• **Data Science**: Jupyter, TensorFlow<br>• **Game Dev**: Godot Engine |

## ⚡ Installation

**Just one app to start!**

1. Download and install **FluxLinux** (APK below).
2. Open the app and follow the **Setup Wizard**.
3. *That's it!* key components (like Termux) and your Linux environment are configured automatically within the app.
