# Flux Linux v1.4 Release Notes

### 1. Advanced GPU Acceleration (Turnip/Zink drivers)
A significant focus was placed on improving GPU acceleration, particularly for snapdragon devices:
- **Major Driver Update**: Switched to `lfdevs/mesa-for-android-container` Turnip drivers with KGSL support, enabling compatibility with the newest Adreno 8xx GPUs.
- **Improved Exisiting Support**: Updated the Turnip driver to `v25.3.3` to drastically improve support for Adreno 7xx and 8xx series GPUs.
- **Hardware Acceleration fixes**: Added the `MESA_VK_WSI_DEBUG=sw` flag for Turnip inside PRoot environments and created a `/dev/dri` compatibility layer for `vkmark` benchmarking.
- **Compositor Fixes**: Disabled the XFCE4 compositor directly via `xfwm4.xml` to ensure Turnip compatibility and prevent visual glitches or crashes.

### 2. HyperOS Compatibility Enhancements
- Implemented specific fixes for Termux X11 compatibility specifically tailored for Xiaomi's HyperOS, resolving socket access bugs and freezing issues.

### 3. Core Functionality & Environments
- **Root Terminal**: Added support for launching a direct root terminal.
- **Development Setup Fixes**: Fixed issues with Java installations inside the chroot environment and cleaned up the NodeSource repository scripts.
